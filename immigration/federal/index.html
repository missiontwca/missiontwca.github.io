聯邦快速通道（EE）

📢 自2023年6月28日起，正式開始「特定職業類別」、「法語類別」的定向邀請！

👉 近期邀請分數、審查時間請參考「最新消息」內容。

聯邦快速通道（Express Entry，EE）主要針對有經驗的技術工作者，根據各方面（年齡、語言能力、學歷、工作經驗...等）計分後，政府按照候選池中的分數高低送出邀請，允許受邀者提交永居申請。

基本步驟：

  1. 確認資格

    2. EE分為CEC、FSW、FST三個類別，務必先確認自己是否符合任一類別的最低標準（官方工具）。

    • CEC（Canadian Experience Class）
    • FSW（Federal Skilled Worker）
    • FST（Federal Skilled Trades）
    此項目適合有至少一年「加拿大工作經驗」的申請者。
    工作要求：
    • 過去3年中有累計滿1年，可以不連續。
    • 全職（Full-time），或相當同等時數的兼職（Part-time）工作。
    • 屬於 NOC 的「TEER 0、1、2、3類」職業。
    • 工作地點需在「加拿大境內」。
    • 工作時有「合法工作」的簽證，且有「給薪」。
    • ❌ 不可以是自雇（醫師除外）。
    • ❌ 不能是全職學生在讀期間的工作經驗。
    語言要求（英/法文均可）：
    • NOC TEER 0、1類：需達到 CLB 7級。
    • NOC TEER 2、3類：需達到 CLB 5級。
    • 需為2年內取得的有效成績。
    此項目主要適合在加拿大境外的技術人才。

    根據申請人「六大面向」進行評估，滿分100分，申請人要獲得至少67分，方可進入候選池。

    選擇指標：
    1. 年齡（滿分12）
      0
    2. 語言能力（滿分28）

      英/法文均可，最低每科需達到 CLB 7級（ = 雅思6分）。

      0
    3. 學歷（滿分25）

      「大學高度專業（Professional Degree）」指需執業執照的特定科系，即「醫學系、獸醫學、牙醫系、足醫系、驗光系、法律系、脊醫系、藥學系」，以ECA學歷認證結果為準。

      0
    4. 工作經驗（滿分15）
      • 需為「至少一年連續」的全職（Full-time）工作，或相當同等時數的兼職（Part-time）工作。
      • 屬於 NOC 的「TEER 0、1、2、3類」職業，且持續為「同一 NOC」的工作。
      • 加拿大境內外的工作皆可。
      • ❌ 不可以是志工服務或無薪實習，需有「給薪」。
      • 可以是自雇。
      • 可以是學生期間的工作經驗，但需有薪且連續。
      0
    5. 就業安排（滿分10）

      Additional content for Employment Arrangements.

      0
    6. 適應能力（滿分10）

      Additional content for Adaptability.

    財力要求：

    人數包含申請者「本人」、「法定伴侶」、「子女」。如8人以上，每人額外再多 加幣 $3706元。

      0
  2. 計算分數

    3. 如通過以上任一類別的最低標準，即可在EE池中建立檔案。系統會依據綜合排名系統（Comprehensive Ranking System，CRS）計算出總分。

    👉 CRS試算連結
    👉 入池教學
  3. 準備文件

    4. 部份文件所需的取得時間可能較長，建議等待邀請時先行預備。

  4. 受邀申請

    5. 候選池一般約每兩周抽選一次，包含「不限類別」的不定向邀請、針對「特定類別」的定向邀請。若您的分數高於當次分數，則會收到移民邀請信（Invitation to Apply，ITA）。

    ⚠️ 特別注意：通過聯邦移民通道（EE）取得永居者，不可選擇在魁北克省（QC）生活，該省有自己的技術人才移民計劃。
