遞交資料

⚠️ 特別提醒：資料務必詳實，尤其日後有申請移民計劃者，勿省略他國旅居（如：澳洲打工度假）記錄。

📣 好消息！自2023年4月起，良民證可線上申請、郵寄到家，無需再臨櫃領件。
❓ 如被通知需要補件，該如何上傳？請參考「補件」文章。

    收到邀請信後，必須：

    • 「10日」內點選接受，如逾期或拒絕則回到抽選池中
    • 「20日」內上傳文件相關文件
    • 線上刷卡繳費
    • 繳費後「30日」內要進行生物辨識
    • 取得POE後，一年內需入境
  1. 接受邀請

    2. 於中籤後的10日內，登入GCKey並點選「開始申請（Start Application）」。

  2. 填寫資料
  3. 上傳文件（Supporting Documents）
    • 家庭資料（Family Information）

      • 填寫[IMM5645]表格，建議下載並使用Adobe Reader DC開啟編輯，無需簽名。

    • CV

      • 簡短一頁即可，包含現居國家城市、學歷、工作經歷（各工作主要內容）、現職工作。

    • 護照

      • 需包含「個人資料頁」、有「出入境戳章」的所有內頁。

    • 證件照

      • 兩吋（35mm x 45mm）大頭照，依護照規格，像素420x540以上，解析度600以上，檔案大小240kB - 4MB之間，JPEG檔。

    • 體檢證明

      • 上傳體檢後拿到的「Information Sheet」，項目包含基本理學檢查、胸腔X光、抽血、尿液檢查（女性建議避開生理期），報告7-10日會由醫院直接上傳至eMedical系統。

      ⚠️ 如未能在時限內進行體檢，需上傳已預約的證明。

      台灣共有三間醫院可進行：

      台北 - 臺安醫院：

      依時段分為現場排隊及網路預約，請參考[注意事項]。
      地址：台北市松山區八德路二段424號 健診中心1F
      體檢時間：週一至週四 9:30-10:30、13:30-15:30，週五9:30-10:30

      台北 - 馬偕醫院：

      現場抽號碼牌辦理，不接受預約，請參考[注意事項]。
      地址：台北市中山區中山北路二段92號 平安樓健檢中心16F
      體檢時間：週二至週五 13:30-14:45

      台中 - 中國醫藥大學附設醫院：

      需電話預約（04-2205-2121，分機15620-15623），請參考[注意事項]。
      地址：台中市北區育德路2號 急重症大樓後棟8F（五權路與錦新街口）健康醫學中心
      體檢時間：週一至週五 13:00-16:00

    • 良民證

      • 警察刑事紀錄證明書（俗稱良民證），凡曾居住過6個月以上的國家皆需申請。台灣良民證申請方式如下：

      線上申請：以自然人憑證透過MyData平台申請郵寄到家，或於2.5工作天後攜帶應備文件至所選地點領取。
      臨櫃申請：不限戶籍地限制，攜帶身分證至各縣市警察局外事科皆可申請。
      APP申請：下載 警政服務APP，同網路申請方式領取。
    • 補充資料（Optional Documents）

      • 如無則不需上傳。

  4. 線上繳費
  5. 生物辨識

    6. 上傳完資料並繳付款成功後，24小時內會收到一封生物辨識通知（Biometrics Collection Letter），請於30日內到VFS辦理。

    需提前預約（按此），並攜帶以下：

      預約單
      護照正本、2份影本
      生物辨識指示信（Biometric Instruction Letter，列印紙本）
      VFS服務同意書(點此下載)
      地址：台北市信義區松仁路97號（第二交易廣場）7樓A室
      開放時間：週一至週五 09:00 - 17:00

    請提前15分鐘抵達，依照指示拍照、採集指紋，掃描被拒者需30日後才可再次進行。

  6. 等待POE

    7. 如未拒簽或要求補件，約在8週內會收到POE（Port of Entry）信，請注意信上的入境期限（通常為一年），至加拿大邊境換取正式的工籤（參考文章：機場入境流程）。


    2023年各項費用：
    • 體檢：台幣 $4100元
    • 良民證：台幣 $100元，同次第2份起每份台幣 $20元
    • IEC申請費：加幣 $161元
    • 開放式工作簽證費 加幣 $100元
    • 生物指紋辨識費 加幣 $85元
